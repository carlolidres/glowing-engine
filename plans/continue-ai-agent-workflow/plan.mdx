## Risks

- Large pre-existing dirty worktree: avoid source behavior edits and preserve unrelated changes.
- Generated `graphify-out/` is dirty although documented as generated: record as a known cleanup item unless explicitly approved for untracking.
- Historical `version-0-baseline.md` contained older Ant Design language; update only enough to remove contradiction with approved current implementation.

## Rollback Strategy

- All changes are documentation and local planning files.
- No reset, clean, checkout, branch switch, or deletion is required.
- If a documentation edit is wrong, revert that file's diff only.
- If generated-map tracking remains noisy, defer Git index cleanup to a separate approved task.
